Adds related section to all internal Opera pages.

Installation steps:
1. go to "opera://flags" page (requires Opera 16)
2. enable "Extensions on opera:// URLs" option
3. restart Opera
4. download and extract this zip file: https://files.myopera.com/dqdb1/files/RelatedLinks.zip
5. press Ctrl+Shift+E to open Extensions
6. enable Developer Mode in the top right corner
7. click Load unpacked extension button and navigate to the folder containing the extracted files
