Speed Dial Simplifier
Copyright (c) 2013 dqdb

This simple extension removes Stash, Discovery and Google search bar from Speed Dial.

Installation steps:
1. go to "opera://flags" page (requires Opera 16)
2. enable "Extensions on opera:// URLs" option
3. restart Opera
4. install/enable this extension
